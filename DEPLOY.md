# Деплой на Vercel

Проект подготовлен как Vite + React.

## Локальная проверка

```bash
npm install
npm run build
npm run preview
```

## Настройки Vercel

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

## Важно перед полноценным production

Форма обратной связи сейчас имитирует успешную отправку на фронтенде. Для реальной отправки нужно подключить backend/API или внешний сервис формы.
